AR Graffiti

This AR Graffiti prototype explores tangible and gesture-based interaction techniques in augmented reality.

The system includes multiple graffiti interaction methods, including direct placement, path-based interaction, spray painting, coloring, and drag-and-drop manipulation. These interaction modes were designed to investigate how different interaction styles influence user preference, immersion, usability, and creative expression when creating and modifying AR content.

Key Features

  • Multiple AR graffiti interaction techniques:
    • Direct placement
    • Path-based drawing
    • Spray painting
    • Coloring
    • Drag-and-drop manipulation
  • Surface-based AR content placement with spatial alignment
  • Real-time interaction feedback

Design Focus

  • Usability comparison across AR input techniques
  • Designing intuitive creation tools in immersive environments
